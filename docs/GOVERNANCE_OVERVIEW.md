# Governance Overview

NDA THEOBROMA follows a structured governance model separating:

- Normative doctrine
- Execution framework
- Operational baselines
- Technical freeze snapshots

---

## Structural Principles

- Non-decision architecture
- No scoring or predictive outputs
- Educational simulation scope
- Clear separation between doctrine and implementation
- Dated execution baselines
- Verifiable freeze references

---

## Technical Integrity

Technical freeze snapshots are performed with:

- Dated packaging
- SHA256 verification
- Manifest documentation
- Archive preservation

These elements are maintained for institutional traceability and audit readiness.
